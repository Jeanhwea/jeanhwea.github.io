* 常量和变量
  定义常量使用 =const= 关键字，变量有 =var= 和 =let= ，其中 =var= 是旧版本的定义
  变量方法，它没有 scope，但是 =let= 是有 scope 的，所以无论如何 *定义变量请使用
  let 关键字*

  #+BEGIN_SRC javascript
    > if (true) {
    ... var v1 = true;
    ... }
    > v1
    true
    > if (true) {
    ... let v2 = true;
    ... }
    > v2
    Thrown:
    ReferenceError: v2 is not defined
    >
  #+END_SRC
