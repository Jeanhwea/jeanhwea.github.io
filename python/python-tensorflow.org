* 附录
  安装 TensorFlow 之前最好是配合 Anaconda3 发行版一起安装，安装好 Anaconda 过后直接
  使用下面命令安装
  #+BEGIN_SRC sh
    conda install tensorflow==1.13.1
  #+END_SRC

  或者使用 pip 安装
  #+BEGIN_SRC sh
    conda install tensorflow==1.13.1
  #+END_SRC
