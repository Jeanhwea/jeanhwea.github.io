1 Maven 工具简介

Maven 是一个非常优秀的 Java 项目中代码包管理工具。它默认将公共仓库放在 ~/.m2/repository 目录下面。有时候下载 Maven 是需要配置默认下载镜像到阿里云， 这时候需要编辑 ~/.m2/settings.xml 文件。

1.1 配置镜像源

在国内访问官方 Apache Maven Repository 的镜像站比较慢，添加下面的代码可以快速 添加阿里云作为镜像源。 

<mirror>
  <id>aliyun</id>
  <mirrorOf>*</mirrorOf>
  <name>ALiYun Public Repository</name>
  <url>https://maven.aliyun.com/nexus/content/groups/public</url>
</mirror>

2 使用 Maven

Maven 官方给出了一个五分钟教程，里面有详细的介绍。其中我觉得一个比较重要的命令 就是新建一个 maven 项目，这里先粘贴过来。

2.1 常用命令

  • mvn compile : 编译项目
  • mvn test : 运行测试
  • mvn package : 打包项目
  • mvn clean : 清除构建
  • mvn install : 安装文件本地代码仓库
  • mvn site : 生成项目的所有文档

2.2 如何创建项目

mvn archetype:generate -DinteractiveMode=false \
  -DgroupId=io.github.jeanhwea -DartifactId=mapp \
  -DarchetypeArtifactId=maven-archetype-quickstart
  • -DinteractiveMode=false 设置不用交互
  • -DarchetypeArtifactId=... 设置骨架项目

2.3 安装 jar 文件到本机仓库

使用 mvn install:install-file 可以快速安装 jar 包到本地仓库中 

mvn install:install-file -Dpackaging=jar \
  -DgroupId=com.oracle -DartifactId=ojdbc6 -Dversion=11.2.0.4 \
  -Dfile=<path-to-jar>

2.4 查看项目的所有依赖

# 树形显示所有依赖
mvn dependency:tree
# 列表显示所有依赖
mvn dependency:list
