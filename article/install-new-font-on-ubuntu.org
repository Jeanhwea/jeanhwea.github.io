* 安装寻找
  首先需要准备字体，可以去开源网站上找 ttf 格式的字体文件，如果需要 Windows 系统字
  体应该在 ~C:\Windows\Fonts~ 文件夹底下

* 准备和安装
  将需要安装的文件复制到字体文件夹中
  #+BEGIN_SRC sh
    # 准备字体文件
    mkdir /usr/share/fonts/my/
    cp fonts/* /usr/share/fonts/my/
    cd /usr/share/fonts/my
    # 创建字体缓存
    fc-cache .
  #+END_SRC

* 查看字体
  查看安装好的字体
  #+BEGIN_SRC sh
    fc-list
  #+END_SRC

  可以使用 grep 配合查看是否存在某一字体
  #+BEGIN_SRC sh
    fc-list | grep -i wenquan
  #+END_SRC
