* 问题
  别人给发的 Windows 下压缩的 zip 包在 Mac 下解压出来的往往是乱码的，大概如下图所示

  [[file:../static/image/2019/09/unreadable-characters.png]]

* 解决方法
  分析可能是 Windows 默认使用 GBK 编码，到 Mac 下不认，所以安装 unar 工具，使用 GBK 方式
  解压缩包就好

  #+BEGIN_SRC sh
    # 安装 unar 工具
    brew install unar

    # 解压命令
    unar -e GBK data.zip
  #+END_SRC
