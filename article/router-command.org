* 显示路由表
  打印路由表
  #+BEGIN_SRC shell
    route -n
  #+END_SRC

* 添加路由
  添加默认路由
  #+BEGIN_SRC shell
    route add default gw 192.168.1.1
  #+END_SRC

  添加一条普通路由项
  #+BEGIN_SRC shell
    route add 192.168.1.0/32 gw 192.168.1.1
  #+END_SRC

* 删除路由
  删除默认路由
  #+BEGIN_SRC shell
    route del default gw 192.168.1.1
  #+END_SRC
