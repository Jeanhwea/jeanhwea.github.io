1 安装准备工具

直接 apt-get 安装就行 

sudo apt-get install git vim curl

2 安装 RVM

参考 RVM 官网的教程。先安装 RVM 的 gpg 秘钥 

gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3

然后通过 curl 来安装稳定版的 RVM。 

\curl -sSL https://get.rvm.io | bash -s stable

3 使用 RVM 配置 ruby 和 gem

先安装 RVM 在 Ubuntu 环境下的准备环境，并使用 RVM 安装最新版本的 ruby 。 

rvm requirements
rvm install ruby

将默认版本（最新稳定版本）的 ruby 应用在当前系统中 

rvm use ruby --default

设置 ruby 的 gem 环境 

rvm rubygems current

4 配置 RubyGems 的镜像

国内的 Gem 镜像通常是被墙了，所以可以使用 Ruby China 的 Gem 镜像来改善国内的 gem 下载速度。

更新 gem 系统 

gem update --system # 这里请翻墙一下

替换 gem 镜像源 

gem sources --add https://gems.ruby-china.org/ --remove https://rubygems.org/

5 使用 Gem 安装 rails

直接安装 rails 就可以了 

gem install rails

6 建立 rails 工程

例如建立一个博客工程 blog 

rails new blog

启动 rails 服务器 

cd blog && rails server

打开浏览器访问想要的网址即可，结束。
