1 Color Hunter

Color Hunter 是个十分直接的网站，最大的特点就是提供了很多现成的配色方案，数 量也非常多，对配色小白很友好，简单易上手。你只需要浏览并选择你喜欢的配色即可。

2 Picular

Picular 是一个色彩搜索网站，打开网站首页，界面非常清爽，左侧是搜索框，右侧则是 瀑布流式的色彩卡片。

3 UI Gradients 和 Web Gradients

渐变色在 PPT、Ai 中很常用, UI Gradients 和 Web Gradients 网站提供了直观的渐变 色方案，整体风格柔和干净。使用时只需在 PPT 中选择渐变填充，然后使用取色器吸收 两端颜色即可。

4 中国色

中国色网站中，当你选择一个颜色后，网页背景会根据该色彩改变，颜色的 RGB 数值也 会在网页中展示出来。这两个网站能够让你 feel 到日式小清新配色，还有中国色彩的古 典之美。记录下这些数值，一个传统色彩的配色方案你就 get 到啦。
