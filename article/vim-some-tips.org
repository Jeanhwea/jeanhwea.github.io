* 使用超级用户权限保存编辑过的文件
  在不退出 Vim 的情况下，使用超级用户保存编辑好的缓冲区
  #+BEGIN_SRC text
    :w !sudo tee % > /dev/null
  #+END_SRC

* 替换 Windows 换行符为 Linux 换行符
  在 vim 中可以直接使用命令替换字符串，注意 ~^M~ 是按键序列 ~Ctrl-V Ctrl-M~, 不
  是两个字符 ~^~ 和 ~M~
  #+BEGIN_SRC text
    :%s/^M//g
  #+END_SRC

  批量修改多个文件，删除回车符号 ~\r~, 可以使用如下命令直接删除 ~\r~ 字符
  #+BEGIN_SRC sh
    find . -type f | xarge -I {} sed -i -e 's/\r//g' {}
  #+END_SRC
